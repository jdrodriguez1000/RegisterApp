# Style Guide - RegisterApp

Este documento define la identidad visual y los patrones de diseño para la aplicación RegisterApp, basados en la estética moderna y limpia de la interfaz de referencia.

## 1. Paleta de Colores

La aplicación utiliza un esquema de colores vibrante pero equilibrado, orientado a la salud y el bienestar.

| Categoría | Color | Hex (Estimado) | Uso |
| :--- | :--- | :--- | :--- |
| **Primaria** | Blue Accent | `#4A90E2` | Botones de acción, barras de progreso, resaltados. |
| **Fondo Principal** | Off-White | `#F8F9FB` | Fondo de la aplicación. |
| **Superficies** | White | `#FFFFFF` | Tarjetas (Cards), modales y contenedores. |
| **Texto Principal** | Dark Charcoal | `#1A1A1A` | Títulos y cuerpo de texto importante. |
| **Texto Secundario** | Muted Gray | `#757575` | Descripciones, etiquetas y texto de apoyo. |
| **Acento 1** | Vibrant Orange | `#FF9F43` | Resaltados secundarios, categorías nutricionales. |
| **Acento 2** | Fresh Green | `#28C76F` | Indicadores positivos, etiquetas de alimentos. |
| **Oscuro** | Deep Black | `#121212` | Botones primarios de alto contraste y barras de navegación. |

---

## 2. Tipografía

Se debe utilizar una fuente Sans-Serif moderna (preferiblemente **Inter**, **Outfit** o **SF Pro**).

- **Títulos (H1):** Bold, 24px - 32px. Tracking ligero. Color: `#1A1A1A`.
- **Subtítulos (H2):** Medium/Semi-Bold, 18px - 22px. Color: `#1A1A1A`.
- **Cuerpo de texto:** Regular, 14px - 16px. Line height: 1.5. Color: `#757575`.
- **Labels/Etiquetas:** Bold/Semibold, 12px. Uppercase ocasional para énfasis.

---

## 3. Componentes y Estilos

### 3.1. Tarjetas (Cards)
- **Border Radius:** `20px` a `24px` para un aspecto amigable y moderno.
- **Sombra:** Suave y difusa (`box-shadow: 0 4px 20px rgba(0,0,0,0.05)`).
- **Padding:** Generoso (`16px` a `24px`).

### 3.2. Botones
- **Botón Primario:** Fondo Deep Black o Blue Accent. Bordes redondeados (pill-shape o `12px`). Texto en blanco.
- **Botones con Icono:** Círculos o rectángulos redondeados con iconos lineales simples.
- **Interacción:** Hover con cambio sutil de opacidad o elevación.

### 3.3. Entradas de Texto (Inputs)
- Fondo gris muy claro o blanco con borde sutil.
- Bordes redondeados (`12px`).
- Placeholder en `Muted Gray`.

### 3.4. Gráficos y Visualización
- **Barras de Progreso:** Bordes redondeados al 100%. Uso de gradientes sutiles.
- **Arcos/Progress Rings:** Grosor medio, terminaciones redondeadas.

---

## 4. Iconografía
- Estilo: **Linear** (Línea delgada).
- Color: Coherente con el texto secundario o el color de acento según el contexto.
- Tamaño base: `24px`.

---

## 5. Layout y Espaciado
- **Padding Global:** `20px` en dispositivos móviles.
- **White Space:** Priorizar el uso de espacio en blanco para evitar la saturación visual.
- **Alineación:** Limpieza visual con alineación consistente a la izquierda para textos y central para elementos destacados.

---

## 6. Imágenes y Fotografía
- Las imágenes deben ser de alta calidad y resolución.
- Recortes: Uso de bordes redondeados (`20px+`) o formas circulares para mantener la coherencia con los componentes.
- Estilo: Fotografía brillante, saturada y con buena iluminación.
